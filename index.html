Grammar: A / An vs Some

How do we ask for food?

"I am cooking an egg." (Countable: 1, 2, 3 eggs).

"She is slicing some bread." (Uncountable: You can't say "1 bread").

Select the correct rule to unlock the system:

RULE UNLOCKED!

Correct! We use A / AN for countable things (an apple) and SOME for uncountable masses/liquids (some water, some bread).

Is "Milk" countable or uncountable? (Do you say "A milk" or "Some milk"?)
